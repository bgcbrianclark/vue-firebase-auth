# Vue/Firebase with Firestore and Authentication

## Project setup
```
npm install
```
1. Add firebase.json with firebase hosting settings
2. Add /fbConfig.js, `export const fbConfig` with firebase config


### Compiles and hot-reloads for development

npm run serve
